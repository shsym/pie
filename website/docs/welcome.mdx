# Welcome to Pie

:::caution Research Prototype
Pie is a **research prototype** under active development. APIs and features may change without notice.
:::

**Pie** is a programmable LLM serving system. Instead of sending a prompt to a black-box API and waiting, you write **inferlets** — sandboxed WebAssembly programs that run *inside* the engine with full access to the KV cache, token stream, and forward pass. The result: agentic apps and complex reasoning strategies are both **easier to build** and **faster to serve**.

## Try it in 60 seconds

```bash
pip install "pie-server[metal]"     # macOS — see Getting Started for cu126/cu128/vllm/sglang
pie config init
pie serve
```

Then in another terminal:

```bash
pie run text-completion --prompt "Write a haiku about Rust."
```

That's it — the engine loaded a model, scheduled the request, and ran a built-in inferlet end to end.
