# Quickstart

From zero to generated text in under five minutes.

## Prerequisites

- Pie installed ([Installation](./index.mdx))
- A model downloaded (`pie model download Qwen/Qwen3-0.6B`)
- A config in place (`pie config init`)

## Your first prompt

```bash
pie run text-completion --prompt "The capital of France is"
```

You'll see something like:

```
╭─ Pie Run ───────────────────────────────────────╮
│ Inferlet     text-completion@0.1.0              │
│ Model        default (Qwen/Qwen3-0.6B)          │
│ Driver       native                             │
│ Device       cuda:0                             │
╰─────────────────────────────────────────────────╯

The capital of France is Paris, which is located in…
```

## What the command does

```bash
pie run text-completion --prompt "The capital of France is"
```

| Part | Meaning |
|------|---------|
| `pie run` | Boot a one-shot engine, run an inferlet, exit. |
| `text-completion` | The inferlet to run. Resolved to `text-completion@<latest>` via the registry. |
| `--prompt "…"` | Becomes `{"prompt": "…"}` in the inferlet's input JSON. |

`pie run` flags after the inferlet name are folded into the input dict
(int/float/bool inference happens automatically). Add a few:

```bash
pie run text-completion \
  --prompt "Write a haiku about pie" \
  --max-tokens 64 \
  --temperature 0.7
```

## Inspect or edit the config

```bash
pie config show              # pretty-print ~/.pie/config.toml
pie config set server.port 9090
pie config set model.default.hf_repo Qwen/Qwen2.5-3B-Instruct
```

The config schema is documented in the [CLI reference](../cli#configuration-file).

## Change the model

```bash
pie model download Qwen/Qwen2.5-3B-Instruct
pie config set model.default.hf_repo Qwen/Qwen2.5-3B-Instruct
```

## Run a local build

If you've built a custom inferlet, point `pie run` at the `.wasm` and its
manifest:

```bash
pie run \
  --path ./target/wasm32-wasip2/release/my_inferlet.wasm \
  --manifest ./Pie.toml \
  --prompt "hello"
```

## Pin a version

```bash
pie run text-completion@0.1.0 --prompt "hi"
```

Without `@`, Pie asks the registry for the latest version.
